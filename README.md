# Personal Portfolio Website

A clean, responsive personal portfolio website built using **HTML, CSS, and JavaScript** to showcase my projects, skills, and online presence.

This site highlights my work as a Python developer and serves as a central place to explore my projects and connect with me.

---

## Features

- Modern, minimal dark-themed design  
- Responsive layout for desktop and mobile  
- Smooth scrolling navigation  
- Interactive hero section with call-to-action  
- Project cards with direct GitHub repository links  
- Contact section with social media icons  
- Lightweight and fast (no frameworks)

---

## Tech Stack

- **HTML5** – Structure  
- **CSS3** – Styling & responsiveness  
- **JavaScript** – Smooth scrolling and interactions  
- **Font Awesome** – Social media icons
